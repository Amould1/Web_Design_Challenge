Summary: Latitude v. X       Visualizations

The objective of this project was to see if there were any weather trends to be found as you go closer to the equator. I used OpenWeatherMapy Api calls to recieve weather data from approximatly 500 cities. The basis for the trends were max temperature, wind speed, cloudiness, and humidity. Then I used MatPlotLib to plot all of the trends versus latitude.
